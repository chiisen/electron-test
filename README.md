# electron-test
Electron 的介紹與初始設定
開源網址: https://github.com/electron/electron

# 啟動專案
```
// 生成 package.json 設定檔案
npm init

// 安裝 electron
npm install --save-dev electron
npm i

// 執行
npm start
```

# 打包工具專案
```
// 安裝 electron-builder
npm i -D electron-builder

// 執行打包
node build.js
```
